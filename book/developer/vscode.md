# vscode

## vscode中选中多行多光标进行操作及一些常用的命令(快捷键)

- ctrl+shift+p  调出命令面板
- alt+鼠标左键  选中多行同时编辑
- ctrl+shift+L  选中编辑代码中相同的内容
- 选中多行,按tab键可统一向右移动
- 选中文本后,Ctrl  +  [      和   Ctrl  +  ]     可实现文本的向左移动 和 向右移动
- 按住Ctrl + Alt，再按键盘上的上或下键，可以使一列上出现多个光标。
- 选中一段文字，按shift+alt+i，在每行末尾都会出现光标
- 按shift+alt，再使用鼠标拖动，也可以出现竖直的列光标，同时可以选中多列。
- 按ctrl+f，可以搜索当前页面
